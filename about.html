About Doodle LED

Doodle LED is a free, professional-grade web application designed for AV professionals, event planners, system integrators, and display technicians who need to design and configure LED video wall installations.

What is a Video Wall?

A video wall is a large display system composed of multiple LED panels or screens arranged in a grid to create one cohesive visual display. These installations are commonly used in:

  • Corporate environments - Lobbies, conference rooms, and control centers
  • Retail spaces - Digital signage and advertising displays
  • Live events - Concerts, conferences, and trade shows
  • Broadcasting studios - News sets and production facilities
  • Sports venues - Stadiums and arenas
  • Control rooms - Security, transportation, and emergency management

Why Use This Tool?

Designing video wall configurations manually can be time-consuming and error-prone. Our tool streamlines the process by providing:

Visual Planning

See your video wall design in real-time as you add and arrange panels on an infinite canvas.

Signal Management

Track which panels connect to which video sources with our 10-signal color-coding system.

Multiple Formats

Support for both 16:9 rectangular panels and 1:1 square LED modules to match your hardware.

Array Tools

Quickly create large grids of identical panels with our array expansion feature.

Key Features

Panel Types

Choose between two standard LED panel aspect ratios:

  • 16:9 Rectangular Panels - Standard widescreen format used in most professional displays
  • 1:1 Square Panels - Popular for modular LED walls and creative installations

Note: You can only use one panel type per design to ensure configuration consistency.

Signal Routing

Our cable management system helps you:

  • Draw connection chains between panels
  • Assign signals using 10 distinct color codes
  • Visualize data flow paths through your video wall
  • Track which panels share video sources
  • Calculate total pixel resolution and panel counts per signal

Design Tools

Professional-grade features include:

  • Move Tool - Drag and reposition panels with magnetic snapping
  • Array Tool - Create rectangular grids of panels instantly
  • Cable Drawing - Connect panels with visual connection lines
  • Signal Painting - Assign signals to entire chains at once
  • Cable Eraser - Remove connections selectively or clear all at once
  • Undo/Redo - Full history tracking (50 states)

Advanced Features

For power users, we offer:

  • Custom panel dimensions input (width and height in pixels)
  • Total resolution calculator showing combined pixel count
  • Signal usage analytics with panel and pixel totals
  • Infinite workspace with zoom and pan controls
  • Collision detection to prevent overlapping panels

Who Should Use This Tool?

AV System Designers can use this tool during the planning phase to visualize installations, calculate signal requirements, and communicate designs with clients.

Event Production Teams can design stage configurations, plan LED wall layouts for concerts or conferences, and coordinate with technical crews.

System Integrators can spec out installations, plan cable runs, calculate processing requirements, and generate configuration documentation.

Facility Managers can plan permanent video wall installations for control rooms, lobbies, or digital signage networks.

Students and Educators in AV technology programs can learn video wall design principles and practice configuration planning.

Technology

Doodle LED is built with modern web technologies including React for the user interface and HTML5 Canvas for rendering. The application runs entirely in your browser with no server-side processing, ensuring your designs remain private and the tool stays fast and responsive.

Open Source & Free

This tool is completely free to use with no registration, no watermarks, and no limitations. We believe professional design tools should be accessible to everyone in the AV industry.

Doodle LED Tutorial

This comprehensive guide will walk you through every feature of the Doodle LED tool, from placing your first panel to managing complex signal routing across large installations.

Getting Started

Understanding the Interface

The Doodle LED interface consists of three main areas:

  • Left Sidebar (Toolbox) - Contains tools, products, and signal colors
  • Center Workspace - The infinite canvas where you design your video wall
  • Right Panel (Instructions) - Quick reference guide (can be hidden)

Step 1: Choose Your Panel Type

Before placing any panels, decide which format you'll use for your entire video wall:

  1. 16:9 Panels - Standard widescreen rectangular panels (480×270 pixels by default)
  2. 1:1 Panels - Square LED modules (270×270 pixels by default)
Important: Once you place the first panel, you cannot switch to the other panel type until you delete all panels. This ensures your design maintains consistent dimensions throughout.

Step 2: Place Panels on the Canvas

Adding Individual Panels

  1. In the left sidebar, find the "Products" section
  2. Click and drag either the 16:9 or 1:1 panel onto the workspace
  3. Drop it anywhere on the canvas - it will automatically center on your cursor
  4. The first panel placement determines your panel type for the entire design
Pro Tip: Panels automatically snap to adjacent panels when you drag them close. This magnetic snapping ensures perfect alignment without manual positioning.

Using the Array Tool for Grids

To quickly create large rectangular grids:

  1. Select the Edit Array tool from the Tools section
  2. Click on any existing panel
  3. An array panel will appear showing:
    • Columns - Use +/- buttons to set horizontal count
    • Rows - Use +/- buttons to set vertical count
  4. Watch the blue preview outlines show your grid layout in real-time
  5. Click Apply to create the grid, or click outside to cancel
Note: The array builds UP and to the RIGHT from your selected panel (bottom-left origin). The preview shows exactly where panels will be placed.

Step 3: Arrange Your Layout

Moving Panels

  1. Select the Move tool (top of the Tools list)
  2. Click and drag any panel to reposition it
  3. Panels will snap to adjacent panels automatically
  4. Magnetic snapping prevents overlaps and ensures perfect alignment

Deleting Panels

  • Right-click on any panel to delete it
  • Connected cables will be automatically removed
  • Use Ctrl+Z to undo if you delete by mistake

Navigating the Canvas

  • Zoom: Use mouse scroll wheel or the +/- buttons (top-left)
  • Pan: Hold middle mouse button and drag, or hold spacebar and drag
  • Reset View: Click the ⟲ button to center and reset zoom

Step 4: Connect Panels with Cables

Drawing Connections

  1. Select the Add Cable tool
  2. Click on a panel to start drawing
  3. Move your mouse over adjacent panels - they'll highlight as you hover
  4. Click additional panels to extend the chain
  5. Release the mouse to finish the chain
Visual Feedback: As you draw, a blue line follows your cursor showing the path. Panels in your current drawing path are highlighted with a white border.

Chain Rules

  • You can connect up to 5 panels in a single chain
  • Chains show as blue connection lines between panel centers
  • A single isolated panel can exist as a "single-unit chain"
  • Extending an existing chain inherits its signal color automatically

Step 5: Assign Signals

Using the Signal Bucket Tool

  1. Select the Signal Bucket tool
  2. Choose a signal color from the palette that appears (10 colors available)
  3. Click on any panel in a chain to paint the entire chain that color
  4. The signal color indicates which video source feeds that chain
Signal Colors: The 10 signal colors alternate between warm and cool tones for easy visual distinction. Use different colors for different video sources in your installation.

Smart Signal Inheritance

  • When you connect two chains with the Add Cable tool, the first chain's color takes precedence
  • The second chain automatically adopts the first chain's signal color
  • This ensures consistent signal assignment across merged chains

Step 6: Remove or Modify Cables

Remove Cable Tool

  1. Select the Remove Cable tool
  2. Click and drag over cables to erase them
  3. Panels that become isolated automatically become single-unit chains
  4. Signal colors are preserved on isolated panels

Clear All Cables

When the Remove Cable tool is selected, a red banner appears at the top:

  1. Click "Clear All Cables" to remove ALL connections
  2. All panels remain in place - only cables are removed
  3. This is useful for redesigning signal routing without recreating your layout
  4. Use Ctrl+Z to undo if needed

Step 7: Track Your Design

Signal Usage Panel

The left sidebar shows real-time statistics:

  • Per Signal: See how many panels use each signal
  • Unit Counts: Total number of panels assigned to each signal
  • Pixel Counts: (Advanced Mode) Total resolution for each signal
  • Total Panels: Overall count at the bottom

Advanced Mode

Enable Advanced Mode for professional features:

  1. Find the "Advanced Mode" toggle in the left sidebar
  2. Click to enable additional options:
    • Panel Width - Customize horizontal pixel count
    • Panel Height - Customize vertical pixel count
    • Total Resolution - See combined pixel dimensions
    • Pixel counts per signal - Calculate processing requirements
Total Resolution Calculation: The tool automatically calculates your wall's total pixel count by measuring the grid dimensions and multiplying by your panel resolution.

Step 8: Use Undo/Redo

Full history tracking is available for all actions:

  • Undo: Press Ctrl+Z or click the ↶ button (top-left)
  • Redo: Press Ctrl+Y or click the ↷ button
  • History Depth: Last 50 actions are saved

Everything can be undone:

  • Adding or deleting panels
  • Drawing or removing cables
  • Painting signals
  • Creating arrays
  • Moving panels

Common Workflows

Designing a Simple 3×2 Video Wall

  1. Drag a 16:9 panel onto the workspace
  2. Select Edit Array tool and click the panel
  3. Set Columns: 3, Rows: 2
  4. Click Apply - you now have a 6-panel wall
  5. Select Add Cable tool and draw connections left-to-right, top-to-bottom
  6. Select Signal Bucket, choose Signal 1, click any panel to assign

Multi-Source Video Wall

  1. Create your panel layout using arrays or manual placement
  2. Use Add Cable to create separate chains for each video source
  3. Example: Left side = Signal 1, Right side = Signal 2
  4. Use different signal colors to visually distinguish sources
  5. Check Signal Usage panel to verify panel distribution

Redesigning Signal Routing

  1. Select Remove Cable tool
  2. Click "Clear All Cables" in the top banner
  3. All panels remain, all cables removed
  4. Redraw your cable chains with the new routing
  5. Reassign signals as needed

Keyboard Shortcuts

  • Ctrl+Z - Undo
  • Ctrl+Y or Ctrl+Shift+Z - Redo
  • Spacebar - Hold to pan the canvas
  • Mouse Wheel - Zoom in/out
  • Middle Mouse - Click and drag to pan
  • Right Click - Delete panel

Tips and Best Practices

  • Start with layout first: Place all panels before drawing cables
  • Use arrays: Much faster than placing panels individually
  • Plan signal flow: Decide on cable routing before connecting panels
  • Color coding: Use consistent signal colors for source types (e.g., Signal 1 = live camera feeds)
  • Save often: Take screenshots of your design for documentation
  • Advanced Mode: Enable for pixel calculations and custom panel sizes
  • Grid alignment: Trust the magnetic snapping - it ensures perfect alignment

Troubleshooting

Can't place a different panel type

You can only use one panel type per design. Delete all panels to switch types.

Panels won't snap together

Make sure you're using the Move tool and dragging panels close enough to trigger the magnetic snap.

Cable won't connect

Ensure you're using the Add Cable tool and clicking directly on panels (not empty space).

Signal won't change color

Select Signal Bucket tool, choose your color from the palette, then click on the chain.

Array preview not showing

The preview appears when the array panel is open. Adjust rows/columns to see updates.
